安装Hugo

点击进入Hugo的官方Github项目地址，找到hugo_X.XX.X_Windows-64bit.zip点击下载。

解压，将hugo.exe添加到环境变量。

输入hugo version，若出现如下输出则成功。

部署到 Github

git remote origin add
git add .
git commit -m "init blog"
git push origin master

使用 Github Actions 自动部署

name: Hugo Deploy  # 名称随意修改

on:
  push:
    branches: [ master ]  # 修改为你的branch

jobs:
  build:
    runs-on: ubuntu-latest  # 可改为特定版本

    steps:
      - uses: actions/checkout@v2  # 更新
        with:
          submodules: true  # 更新主题
          fetch-depth: 1

      - name: Setup Hugo  # 安装Hugo
        uses: peaceiris/actions-hugo@v2
        with:
          hugo-version: 'latest'  # 可改为特定版本

      - name: Build Hugo  # Hugo生成页面
        run: hugo

      - name: Deploy Hugo  # 部署页面
        uses: peaceiris/actions-gh-pages@v3
        with:
          personal_token: ${{ secrets.ACCESS_TOKEN }}  # 名字与secrets里的对应
          external_repository: backsided/backsided.github.io  # 指定发布的仓库
          publish_branch: master  # 指定发布的branch
          publish_dir: ./public  # 指定发布的目录
